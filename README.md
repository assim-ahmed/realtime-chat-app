# 💬 Real-Time Chat Application

تطبيق محادثة فورية (Full-Stack) مبني باستخدام تقنيات حديثة لضمان سرعة الأداء وتحديث البيانات لحظياً.

---

## 🚀 التقنيات المستخدمة (Tech Stack)

| الجزء | التقنية | الوظيفة |
| :--- | :--- | :--- |
| **Frontend** | React.js (Vite) | بناء الواجهات التفاعلية |
| **State Management** | Zustand | إدارة حالة التطبيق والسوكيت بشكل مركزي |
| **Styling** | Tailwind CSS | تصميم عصري ومتجاوب (Responsive) |
| **Backend** | PHP | معالجة البيانات والـ API |
| **Real-time** | Ratchet (WebSockets) | الاتصال اللحظي بين المستخدمين |
| **Database** | MySQL | تخزين بيانات المستخدمين والرسائل |
| **Security** | JWT | تأمين المسارات وتسجيل الدخول |

---

## ✨ المميزات الحالية (Core Features)

* ✅ **Real-time Messaging:** إرسال واستقبال الرسائل فوراً.
* ✅ **User Presence:** إظهار حالة المستخدم (Online/Offline) لحظياً.
* ✅ **Global Store:** إدارة الـ WebSocket من خلال Zustand Store لضمان استقرار الاتصال.
* ✅ **Authentication:** نظام تسجيل دخول محمي بـ JWT وتشفير كلمات السر.
* ✅ **Responsive UI:** تصميم مهيأ للعمل على كافة أحجام الشاشات.
